16 tools.
One workflow.

Every tool is purpose-built for jewellery modelling in Rhino 3D. From automated pave stone placement to layer-based BOM reports, the plugin handles the repetitive and technical work — so you can focus on design.

Updated continuously. New tools and enhancements ship with each release.

01
stone_setting
Auto Pave Pro
Automates pave stone placement across surfaces and solids. Handles spacing, orientation, and pattern distribution with adjustable parameters.
↗
02
stone_setting
Pave Planner Pro
Plans and builds pave metal geometry with surface-adaptive offset generation for prong and seat construction.
↗

stone_setting
Single Line Pave
Places stones in a precise single-row arrangement along any path. Ideal for eternity bands and straight-line pave designs.
↗

Curves to Diamonds
Distributes diamond geometry along any curve. Control count, spacing, rotation, and size for production-ready results.
↗

Places diamonds guided by a curve projected onto a surface. Follows surface contour for accurate placement on complex forms.
06

Diamond Report By Circles
Generates a structured diamond report by reading circle geometry. Outputs stone counts and sizes grouped by layer.
07
cutting
Azure Cutter Pro
Cuts precise stone seats into metal geometry. Designed for clean boolean operations producing production-ready setting geometry.
↗
08
layers_reports
Layers for Jewellers
A standardised layer setup for jewellery modelling. Creates and manages all common jewellery layers in one step.
09
layers_reports
Layer BOM Report
Generates a Bill of Materials from your layer structure. Exports object counts and names for accurate production documentation.
10
layers_reports
Group Objects by Layer
Organises objects into named groups matching their layer structure. Simplifies complex models for downstream use.
11
layers_reports
Group Objects by Color
Groups objects based on display colour. Useful where colour encodes material type or component role across the model.
12
layers_reports
Sync Object Color to Layer
Synchronises all object display colours to match their parent layer colour. Maintains visual consistency in one click.
13
layers_reports
Auto Layer by ObjectType
Automatically assigns objects to appropriate layers based on geometry type. Streamlines model organisation without manual sorting.
14
layers_reports
Randomize Layer Colors
Assigns distinct random colours to all layers. Helps differentiate components visually in complex multi-layer models.
15
utility
Metal Weight Calculation
Calculates estimated metal weight of selected geometry based on material density. Supports common jewellery alloys.
16
utility
Pro Capture
Captures clean viewport renders with configurable output settings. Produces consistent documentation images directly from the viewport.
17
utility
Switch Display
Quickly toggles between display modes for selected objects or the full model. Useful for reviewing geometry at different stages.

Set your parameters — stone size, spacing, layer names, material type — and confirm. Results apply directly to your Rhino model.
