How It Works

From install to production

01
Install the plugin
A single Rhino plugin file. Install via the standard Rhino plugin manager. All tools appear immediately in your command panel and toolbar.
02
Run any command
Access tools by command name or from the MSH toolbar panel. Each tool opens a clean input dialog with all parameters visible.
03
Adjust and confirm
Set your parameters — stone size, spacing, layer names, material type — and confirm. Results are applied directly to your Rhino model.
04
Production-ready output
The geometry, reports, and layer structures produced are ready for manufacturing, rendering, or client presentation without further editing.
